# BlogHub

CRUD com Ruby on Rails.

# Requerimentos

### `Ruby 3.3.3`

### `Rails 7.1.3.4`

### Instalação de Dependências

Instale as dependências do projeto usando Bundler e Yarn:

```bash
bundle install
yarn install
```

### Rodando o Servidor
Para iniciar o servidor Rails, use o comando:

```bash
Copiar código
rails server
```

Abra o navegador e acesse http://localhost:3000 para ver sua aplicação em funcionamento.
